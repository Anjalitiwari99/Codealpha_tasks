# Stock Portfolio Tracker

This is a simple web-based Stock Portfolio Tracker built using Python and Flask.

## Features
- Add stock ticker symbols and quantity
- Calculate total investment value
- Uses predefined stock prices (hardcoded dictionary)
- Stores portfolio data in a CSV file
- Automatically updates total portfolio value

## Technologies Used
- Python
- Flask
- HTML
- CSV File Handling

## How to Run
1. Install Flask
2. Run: python app.py
3. Open http://127.0.0.1:5000 in browser

## Purpose
This project was built to understand basic concepts like dictionaries, file handling, web routing, and dynamic data rendering using Flask.# Codealpha_tasks
